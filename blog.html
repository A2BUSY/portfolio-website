Useful VS Code extensions

  • Microsoft Edge Tools - This extension, allows Microsoft Edge (and console) to be inside VS Code, which minimises switching between browser and IDE to make changes.
  • AutoRename Tag - This extension, allows when changing the first HTML tag, it will also automatically change the second one.
  • Prettier - This extension, automatically formats code.

How to use angle brackets in text in HTML

To use < and > brackets, we are required to type out either either &lt; for "<" or &gt; for ">" in our text, or another solution is to use the <pre> tag which means "preformatted"
